# Cozy Mug Cozy

A quick project to keep drinks warm. Track your stitches and mark your place as you go.

<PatternControls />

## Supplies
- Worsted weight yarn
- 4.0 mm hook
- Tapestry needle

## Pattern
1. `mr`, work 6 `sc` into ring. Join with `sl st`.\
   - [ ] Round complete
2. `ch` 1, *2 `sc` in each stitch around*. (12)\
   - [ ] Round complete
3. `ch` 1, *1 `sc`, `inc`* repeat around. (18)\
   - [ ] Round complete
4. `ch` 1, *2 `sc`, `inc`* repeat around. (24)\
   - [ ] Round complete
5. `ch` 1, *3 `sc`, `inc`* repeat around. (30)\
   - [ ] Round complete

## Finishing
- [ ] `sl st`, fasten off, and weave ends.
- [ ] Wrap around mug and enjoy!
