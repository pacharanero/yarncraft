# yarncraft

Collected crochet reference tables sourced from the Craft Yarn Council standards:

- `docs/crochet-abbreviations.md` — stitch abbreviations for U.S., U.K., and Tunisian crochet plus common measurements.
- `docs/pattern-notation-and-measurements.md` — repeat symbols with a link to measurement abbreviations.
- `docs/us-uk-crochet-terms.md` — pointer to the combined U.S./Canadian and U.K. abbreviation table.
